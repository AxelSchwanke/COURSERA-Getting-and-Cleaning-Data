Code Book
========================================================

Project raw data was downloaded from 
https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip 
on April 08, 2015 from "Human Activity Recognition Using Smartphones Data Set" 
(http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones). 

The dataset contains physical measures which capture motions of a smartphone. 
More information on how the data is collected, can be found at:
http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones#

After downloading,the data was pre-processed:
 1. Merging ot training and testsets to create one common dataset.
 2. Extracting the measurements of mean and standard deviation for each measurement. 
 3. Using descriptive activity names for the activities in the dataset
 4. Appropriately labeling ot the variables in the dataset with descriptive variable or feature (column) names

More information about the detailled processing steps can be found in the file "run_analysis.R".
